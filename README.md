# Promo Mechanics Selector

One-page service for selecting marketing promotion mechanics and preference types for a specific business situation.

The app works as a static site:

- structured business, product, audience, channel, goal, and risk criteria;
- weighted multi-select logic where the first selected option in a criterion is primary;
- ranked mechanics with fit percentage, recommended preference, pros, cons, and usage notes;
- responsive desktop and mobile layout.

Open `index.html` directly or serve the folder with any static HTTP server.
